# LFI SSH Fuzzer - Quick Reference

## Basic Commands
```bash
# Basic scan with prompts
python3 lfi_ssh_fuzzer.py

# Advanced mode with extra features
python3 lfi_ssh_fuzzer.py -adv

# Dry run (test configuration)
python3 lfi_ssh_fuzzer.py --dry-run

# Save results to file
python3 lfi_ssh_fuzzer.py --output scan_results.json
